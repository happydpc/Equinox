How to calculate equivalent stress from STF file

Equivalent stresses can be calculated directly from stress input files (STF). Following video demonstrates how to do this.


For this, select and right-click on a STF file (or multiple STF files), and select Equivalent Stress Analysis from the popup menu as shown in the following figure.

Analysis Options

There are two analysis options as follows;
  • Fast Analysis This option is suitable for performing large number of analyses as the process is 3 to 4 times faster than the detailed analysis. Only equivalent stresses will be generated as outputs and no plot or typical flight information will be generated. Plots can still be generated on demand once the analysis is complete.

  • Detailed Analysis This option is suitable for performing detailed investigations on pilot points as all analysis data such as plots (mission profile, level crossing etc.) and spectrum statistics will be generated.
