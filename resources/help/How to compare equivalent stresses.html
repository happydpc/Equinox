How to compare equivalent stresses

Equivalent stresses can be compared via equivalent stress values or life factors. Following video demonstrates this process.


For this, select the equivalent stresses you want to compare from the file tree, right-click and select Compare Equivalent Stresses from the popup menu as shown in the following figure.

Plotting life factors against mission parameters

Life factors can be plotted against mission parameters assigned to spectra. For this, select basis mission of comparison and select Plot against mission parameters.
