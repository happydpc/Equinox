How to plot level crossing

Once equivalent stresses are calculated, cycle count information is available for creating the level crossing plots. Following video demonstrates how to do this.


For this, right click on an Equivalent Stress item (or multiple items) and select Plot Level Crossing;

This will show the Level crossing input panel.

Design service goal (DSG)

By default, cycle counts are normalized with spectrum validities. Instead, you can also input the DSG for each level crossing. Rainflow cycle counts are scaled with the DSG values for generating the plot.

Series naming

Generated level crossings plot will contain series for each compared spectrum. In order to uniquely identify each spectrum, the name of the series can include A/C program, A/C section, fatigue mission and rainflow omission levels, depending on the selected options.
