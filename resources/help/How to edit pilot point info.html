How to edit pilot point info

You can set a descriptive location image and additional information to pilot point stress files (STF). Following video demonstrates how to do this.


For this, click on a STF file. As you will see, the current pilot point info will be displayed within the Info View. Click on the location image to set an image for the pilot point. You can also enter additional information relevant to the pilot point and save it via clicking the Save button located at the top right corner.
