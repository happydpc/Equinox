Excalibur

Excalibur is a GFEM stress extraction and sorting tool. It utilizes fatigue spectrum - loadcase correlations in order to generate stress input files (STFs). Following sections describe the inputs necessary to run the plugin via Equinox. Click on the following button for a quick demonstration of the plugin in action.


Loadcase correlations

In this section, the fatigue spectrum loadcase correlations are supplied. Following 2 files are expected as inputs;
  • Attributes table (.xls) Attributes table file is similar to a spectrum conversion table, but it also contains additional loadcase information such as load type and criteria.
  • Loadcase keys file (.lck) This file contains EGL loadcase names, loadcase numbers and loadcase family information.
Note that, usually correlation pairs are supplied for a certain A/C section and fatigue mission.

Element stresses

Extracted GFEM element stresses are supplied within this section. First, select the stress type of input files, then browse (or darg & drop) the directory that contains all input stress files. Note that standard ISAMI generated GFEM stress files are the expected file format. Each file should contain stresses for only 1 element.

Pressure

Cabin pressure loadcases are usually not contained within the loadcase correlatons. Therefore, in case delta-p cases are to be included, enter the delta-p loadcase number within the stress files.

Stress sorting criteria

Based on the selected criteria, 1 loadcase will be selected from a lodcase family representing each fatigue event within the correlated spectrum. Stress sorting criteria depends on the input stress type selected in the Element stresses section.

Output directory

Select the output directory where the generated stress input files will be saved. Note that, the following 2 directories will be created within the given output directory;
  • stfs This directory will contain all generated stress input files.
  • logs This directory will contain all generated log files. Note that, no log file wile will be generated if log level is selected as off.

Options

Here, you can select the log level and whether to run the stress sorting tasks in parallel or sequentially.
